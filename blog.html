Difference between map(),forEach(),filter() and find()

The main difference between map() and forEach() is the returning value. map() returns value and forEach () doesn't return value. Their functions are same but returning value remains different. On the other hand,the function of filter() and find() are almost same but the main difference is filter() returns an array containing the element that satisfies the condition, but find() returns the element itself that satisfies the condition.

Difference between cookie ,session storage and local storage

Local storage is useful for storing data that the user will need to access later, such as offline data. Session storage is similar to cookies, but the data is only stored for the current session. This means that the data will be deleted when the user closes the browser.
